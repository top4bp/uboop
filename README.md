# uboop
Boop like app for Ubuntu

## Setup

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd uboop
   ```
2. Create and activate a Python virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the App

Run the main entry point:
```bash
python main.py
```

## Notes
- All dependencies are listed in `requirements.txt` for easy setup.
- The `venv/` directory is excluded from version control via `.gitignore`.
- For development, activate the virtual environment before running or testing code.
